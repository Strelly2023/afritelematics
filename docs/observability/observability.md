# Observability

Observability exposes metrics, traces, logs, chain summaries, and state exports without modifying protocol execution.

## Components

- `runtime/metrics/`
- `runtime/tracing/`
- `runtime/observability/exporter.py`
- `runtime/observability/dashboard.py`

## Exports

- JSON snapshots
- Prometheus-style numeric metrics
- Chain summaries
- Proof batches
- State snapshots
