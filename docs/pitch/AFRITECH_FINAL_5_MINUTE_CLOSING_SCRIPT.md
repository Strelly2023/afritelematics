# AfriTech Final 5-Minute Closing Script

Status: FINAL 5-MINUTE CLOSING SCRIPT
Classification: COMMERCIAL_CLOSE_SURFACE

Purpose: define the final five minutes of the partner conversation after the
live demo has shown replay, signed receipts, external verification, and trust
SLA.

This script is a close-stage commercial surface.
It is not proof that the deal is already won.

## Closing Goal

Leave the meeting with one concrete next action:

- sandbox verification session
- pilot scoping call
- contract review

## Script

Use:

```text
You have now seen the replay, the signed receipt, the external verification
surface, and the live trust SLA. The question is no longer whether the system
can explain one workflow. The question is whether one of your high-friction
workflows is worth verifying this way.

If the answer is yes, the next step is simple: one bounded pilot, one owner,
one success metric, one fixed fee, and one verification path your team can
review independently.

If the answer is no, we should stop here rather than force a platform story.
If the answer is yes, let us schedule the pilot scope review and put the first
contract draft in front of your team.
```

## Close Rule

End with only one ask:

```text
pilot scope review this week
```

Do not end with:

- a vague follow-up
- an open-ended architecture discussion
- a product roadmap brainstorm
