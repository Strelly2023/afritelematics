# AfriTech

AfriTech is a Sovereign Distributed Execution Ledger Protocol.

Core principle:

```text
Execution = Proof = Consensus = Ledger = State
```

## Key Features

- Zero-trust secure network
- Deterministic execution engine
- Verifiable proof system
- Byzantine-resilient consensus
- Trust scoring and slashing
- Immutable execution ledger
- Replay-verifiable state machine
- Read-only state service layer
- Observability and export tooling

## Pipeline

```text
Execution -> Proof -> Consensus -> Block -> State -> Service
```

## Current Deployment Stance

Repository-side protocol validation is available. AfriRide live pilot activation remains gated until live backend, device, replay, and operator gates pass.
