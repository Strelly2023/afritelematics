# AfriRideMobile

GA Elite role-separated mobile scaffold for Rider, Driver, and Operator workflows.

Classification:

```text
role-separated constitutional display surface
legitimacy_effect: 0
authority_effect: 0
```

Authority boundary:

```text
Backend proof decides.
Replay verifies.
API exposes confirmed state.
Mobile displays.
Mobile does not invent truth.
```

This app may:

```text
request backend actions
display confirmed backend state
show replay receipts
expose evidence visibility
support rider, driver, and operator workflows
```

This app must not:

```text
decide ride truth locally
infer replay validity
mutate canonical state without backend confirmation
treat UI state as authority
```

Run locally:

```bash
npm install
npm start
```

Set the backend URL before field testing:

```text
EXPO_PUBLIC_AFRIRIDE_API_URL=https://afriride-api.onrender.com
```
